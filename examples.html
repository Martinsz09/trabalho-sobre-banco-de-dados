Exemplo de Código SQL

      
-- Exemplo de consulta SQL
SELECT * FROM clientes
WHERE idade > 30
ORDER BY nome;

Exemplo de Código NoSQL (MongoDB)

      
db.clientes.find({
  idade: { $gt: 30 }
}).sort({ nome: 1 });
